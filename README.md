# windelectrostation_hackathon
About Windelectrostation Hackathon

_**Инструкцией по установке зависимостей и запуску:**_

1. Установить зависимости: pip install -r requirements.txt
2. Запустить процесс обучения: python train.py
3. Запустить процесс прогнозирования: python predict.py
4. Результат прогноза будет находится в файле: prediction.csv
5. Файл с весами модели располагается по пути: ./models/model.pkl

_**Описание подхода:**_

В данном решении была использована модель градиентного бустинга LightGBM.
Ключевые этапы пайплайна:
1. Генерация признаков: на основе временной метки были извлечены признаки 'hour' и 'dayofweek', так как выработка энергии часто зависит от времени.
2. Разделение данных: для обучения и валидации использовался метод 'train_test_split' (80% на обучение, 20% на валидацию), чтобы корректно применять механизм ранней остановки ('early_stopping').
3. Обучение: Модель обучалась с использованием метрики MAE (Mean Absolute Error). Применялась ранняя остановка, чтобы избежать переобучения.

_**Используемые метрики:**_

MAE (Mean Absolute Error / Средняя абсолютная ошибка): основная метрика для оптимизации модели ('objective: regression', 'metric: mae').
R2 (Коэффициент детерминации): использовалась для дополнительной оценки качества модели на валидационном наборе.

_**RANDOM SEED:**_

Для обеспечения полной воспроизводимости результатов в скрипте 'code/train.py' зафиксирован random seed ('SEED = 42'). 
Это гарантирует, что при каждом запуске обучения на одних и тех же данных будет получена идентичная модель.
